## Project Purpose

This database system helps to:
- Centralize student health records securely
- Manage clinic appointments efficiently
- Track medical inventory and supplies
- Generate health reports and analytics
- Ensure student privacy and data security

## Key Features

- Student health profiles and medical history
- Appointment scheduling and management
- Medical inventory tracking with low stock alerts
- Health reports and visit statistics

## Technologies Used

- **Database**: MySQL
- **Design Tool**: ERD (Entity Relationship Diagram)
